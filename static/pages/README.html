About website:

This project is the new CSDT website that will combine http://csdt.rpi.edu/ and https://community.csdt.rpi.edu/ under one url - http://csdt.rpi.edu/

The new website is being developed using the bootstrap library, resulting in a site that is mobile device friendly.

Getting Started:

To get started developing on the site, first clone the repository into a local directory:

mkdir website

cd website

git clone https://git.hass.rpi.edu/csdt/website

The site is constructed using embedded javascript (note the .ejs files). You will find header .ejs files containing code that should be applied to each page in a folder, for example Adinkra or Graffiti.

Install Node.js:

The .ejs files are compiled using harp.js. To setup for compiling the site, install node.js available here:

https://nodejs.org/en/

(If working in Windows, restart your machine).

##Install harp.js Next, install harp using the NPM package manager included with node:

npm install -g harp

(this installs harp.js so that it is available 'globally' on your system.)

Compile the site:

To compile the site, cd into the directory where you have cloned it:

cd website

then

harp compile

Harp will compile the site and all of its resources into the www folder:

cd www

and you will see all of the folders and html files that harp generated for deployment.

Start the webserver:

Go back to the root directory for the project:

cd ..

and you can start the web server that comes with node with:

harp server website

In your browser, navigate to http://localhost:9000/ and you should see the site.
