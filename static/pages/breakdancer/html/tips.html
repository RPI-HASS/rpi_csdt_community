Following are some pointers on using this software:

The image below depicts the virtual dancer in its initial state.

Virtual Dancer with Left Arm and Hip rotated by 90 degrees about the Y axis

Rotation of the arm about the y-axis appears to change only the hand.

Similarly, rotation about the y-axis of the leg appears to change only the foot.

Anti-clockwise rotation by +60 degrees

Rotation in the Anti-clockwise direction needs a positive angle.

Clockwise rotation by -60 degrees

Rotation in the Clockwise direction needs negative angle.

Try using the Load Demo before starting your own simulation. This will give you a good feel for the software.

To reset the entire simulation, use the Refresh button of your browser. The Refresh function of the software only resets the current frame.
